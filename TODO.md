# 实现进度

## ✅ v1 基础（已完成）
完整太阳系模拟器：19 类天体自定义 GLSL 着色器 + 交互系统 + GitHub Pages 部署。

## ✅ v2 质量提升（已完成）

### 性能与适配
- **质量三档**：性能/平衡/质量，启动时按 GPU 自动检测；影响分段数、星点数、Bloom 分辨率、像素比、阴影开关。控制面板可手动切换（reload 重建）。
- **移动端响应式**：窄屏控制面板折叠成底部抽屉，齿轮按钮唤出；信息面板与搜索栏自适应。

### 科学性
- **真实轨道力学**：开普勒方程牛顿迭代，椭圆轨道 + 真实倾角(inclination) + 升交点经度(ascendingNode) + 偏心率。冥王星 17° 倾角显著可见。
- **真实尺度切换**：观感模式(默认缩放) ↔ 真实尺度(直径/半长轴对数压缩)，控制面板开关。

### 视觉升级
- **体积感日冕**：太阳 3 层光晕从平面 Sprite 升级为反向法线半透明球壳 + 流动丝缕噪声，任意角度有厚度。
- **行星大气升级**：金星/地球/火星/泰坦大气壳加入流动噪声厚度。
- **地球独立云层**：云从地表着色器剥离为独立半透明球壳，慢速独立旋转，云层飘过地表；夜面云微光。
- **镜头光晕**：太阳十字光芒 Sprite，相机越正对越亮。
- **土星环带阴影**：在土星表面投 MultiplyBlending 暗环（ShaderMaterial 无法 receiveShadow 的务实替代）。

### 交互增强
- **时间轴日期**：右下角按倍率推进的实时模拟日期(从 2026-07-20 起)。
- **成就系统**：6 个成就(找到冥王星爱心/凝视大红斑/土星环/全矮行星/逐日/探索 5 天体)，localStorage 持久化，解锁 toast。
- **键盘导航**：Tab/方向键遍历天体，Enter 聚焦。
- **reduced-motion**：prefers-reduced-motion 时关闭流星、默认暂停时间。

## 🔧 已知限制
- ShaderMaterial 自定义着色器无法自动 receiveShadow（Three.js 限制）。土星环阴影用 MultiplyBlending 暗环务实替代；小行星/命名天体(MeshStandard)在质量档正常 castShadow。
- BokehPass 景深未启用(会牺牲焦点清晰度，与"看清天体"冲突)。
- 真实尺度为对数压缩科普展示，非严格比例。
